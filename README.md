# RiceGeneFormer BIB final package

本仓库现在只保留 RiceGeneFormer / PRSNet 当前 BIB 投稿整理版的轻量交付材料。

## 终版文件夹

请以这个文件夹为准：

`docs/RiceGeneFormer_BIB_FINAL_CLEAN_20260615/`

里面已经整理好：

1. `manuscripts/`：英文和中文终版 Markdown 稿件。
2. `exports/`：英文和中文 Word/PDF 导出稿。
3. `figures/`：7 张终版 SCI/BIB R 图，每张包含 SVG、PDF、TIFF、PNG。
4. `source_data/`：支撑终版图和关键结论的轻量 source data。
5. `audits/`：BIB 风格、审稿风险、数据/代码可用性、数据来源等自查文档。
6. `PACKAGE_MANIFEST.json`：终版文件清单。
7. `README_FINAL_CLEAN_PACKAGE.md`：终版包说明。

## 当前论文定位

本文定位为 Briefings in Bioinformatics 风格的 leakage-aware、ordinal-aware benchmark / diagnostic framework。
不声称 SOTA，不声称跨区域 robustness，不把 gene-attention 结果表述为已验证生物发现。

## 仓库清理原则

- GitHub 只保留终版投稿文件夹、总进展文件和本 README。
- 不上传原始 genotype/phenotype 数据、模型权重、checkpoint、日志、SLURM 输出或临时中间结果。
- 旧版图、旧版 source-data 中间目录、旧稿件草稿和打包 zip 已从仓库版本中清理。
