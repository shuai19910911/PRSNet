# RiceGeneFormer BIB final clean submission package

This folder is the cleaned final handoff for the BIB-framed RiceGeneFormer manuscript.

Claim boundary: benchmark/diagnostic framework only; no SOTA, robustness, or biological-discovery claim.

## Folder contents

- `manuscripts/`: final English and Chinese markdown manuscripts.
- `exports/`: final Word and PDF manuscript exports.
- `figures/`: 7 final R-rendered SCI/BIB figures, each in SVG/PDF/TIFF/PNG.
- `source_data/`: lightweight source data for the final figures and claim checks.
- `audits/`: reviewer-risk, BIB-style, citation/source, and data/code-availability audit documents.
- `PACKAGE_MANIFEST.json`: machine-readable file manifest.

## Excluded on purpose

Raw genotype/phenotype matrices, GWAS arrays, model checkpoints, logs, SLURM outputs, caches, and exploratory intermediate files are intentionally excluded.
